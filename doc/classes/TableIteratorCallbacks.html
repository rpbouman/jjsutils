TableIteratorCallbacks Class

Extends ResultsetIteratorCallbacks
Defined in: src\jdbc\createTableIterator.js:92
Module: TabeIterator

The TableIteratorCallbacks class is a {{#crosslink "ResultsetIterator"}}{{/crossLink}}. The caller can provide all the usual hooks for a plain {{#crosslink "ResultsetIterator"}}{{/crossLink}}, and then pass it to the {{#crosslink "TabeIteratorFactory/createTableIterator:method"}} method.

afterAll

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:226

The afterAll callback is called after iteration. It is always called, even if there were no rows in the resultset.

afterLast

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:214

The afterLast callback is called after all rows have been iterated. It is called only if there was at least one row in the resultset. If it is called, it is called right before the afterAll() callback.

beforeAll

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:165

The beforeAll callback is called before any rows are iterated. It is always called, even if it turns out that there aren't any rows to iterate over.

beforeFirst

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:177

The beforeFirst callback is called right before the first row is iterated. It will only be called if there are in fact rows to iterate.

forEach

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:191

The forEach callback is called for each row in the resultset.

ifNone

()

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:203

The ifNone callback is called only if there are no rows to iterate. ifNone is called after beforeAll(), but before afterAll().

Properties

columns

Unknown

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:93

The columns property is an object that holds metadata that describes the columns of the iterated resultset. The data for this is obtained from the java.sql.ResultSetMetaData object that is associated with the resultset. Keys of the columns object are the column names. Values of the columns object are objects that describe the properties for the result set column as key/value pairs.

row

Unknown

Inherited from ResultsetIteratorCallbacks: src\jdbc\iterateResultset.js:105

The row property is an object that holds the values of the row currently being iterated. Keys of the row property object are the column names of the resultset. Values are obtained by calling java.sql.ResultSet#getObject for the respective column.
