jdbc Module

Defined in: src\jdbc\jdbc.js:40

The jdbc module provides access to java database connectivity (JDBC). Typically you acquire access to the module by naming it as a dependency in a call to define() when defining your own module. The object passed to the define() callback to represent the dependency can be considered a singleton jdbc class, which exposes all its functionality as static methods.

This module provides the following classes:

  • JDBC
  • JDBCConnectionProperties
